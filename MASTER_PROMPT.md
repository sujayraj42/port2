## ◈ CODE STANDARDS

### HTML
- `lang="en"`, proper `<main>`, `<section>` with `aria-label`, `<article>` for cards
- All `data-*` attributes documented
- Popover API for modals: `popover`, `popovertarget`
- `<details>/<summary>` for any accordion
- `role="presentation"` + `aria-hidden="true"` on canvas
- Skip link: `<a class="skip-link" href="#main">Skip to content</a>`

### CSS
- `@layer` everything — zero specificity battles
- `clamp()` everywhere for fluid typography and spacing
- `oklch()` for all custom colors
- `color-mix()` for opacity variants
- CSS nesting — native, no Sass
- Zero `!important` except in `@layer utilities`
- CSS custom properties for ALL values that might change

### JavaScript
- `type="module"` — all files are ES modules
- `export/import` — explicit named exports
- `async/await` + `Promise.all()` for parallel init
- JSDoc on every exported function
- `prefersReducedMotion()` guard on every animation
- `isTouchDevice()` guard on cursor/magnetic/tilt
- `scheduler.postTask()` for background work
- `OffscreenCanvas` + `Worker` for canvas
- Zero `var` — only `const` and `let`
- Zero `querySelector` without null-guard

### Performance
- LCP < 2.5s on 4G
- CLS = 0 (no layout shifts — reserve space for all elements)
- Canvas runs at 60fps on worker thread
- `will-change: transform` on active-only elements
- `content-visibility: auto` on below-fold sections
- `loading="lazy"` if any images
- Preconnect to Google Fonts CDN
