# Sujay Chakravarti — Developer Portfolio

A premium, highly interactive personal portfolio showcasing my journey as a Full Stack Web Developer. Built to demonstrate advanced frontend techniques without relying on heavy frameworks.

🔗 **[Live Demo](https://sujayraj42.github.io/personal-PORTFOLIO/)** (or via Vercel deployments)

## 🚀 Features

- **Skeuomorphic & Dark Mode Aesthetics**: A modern, premium dark theme utilizing custom properties, layered shadow systems, and glowing particle backgrounds.
- **Scroll-Driven Animations (SDA) with Fallbacks**: Native CSS Scroll-Driven Animations where supported; elegant `IntersectionObserver` 0.6s cubic-bezier reveals as a frictionless fallback for unparalleled smoothness.
- **Vanilla JavaScript**: Zero dependencies. All interactivity (loaders, 3-layer custom cursors, view transitions, tilt effects) is powered by pure, modern JavaScript (ES2025+).
- **Web APIs Integration**: Implementation of the HTML `<dialog>`/Popover APIs, Web Workers with OffscreenCanvas for particle rendering, and more.
- **Fully Responsive**: Flawless display across mobile, tablet, and desktop viewports, boasting dynamic layouts via CSS Grid and Flexbox.

## 🛠️ Tech Stack

- **Structure**: Semantic HTML5
- **Styling**: Vanilla CSS3, CSS Variables, Animations (@keyframes, @property), custom Tokens system
- **Logic**: Vanilla JavaScript (ES Module-based architecture)
- **Deployment**: GitHub Pages / Netlify / Vercel

## 📂 Project Structure

- `/assets/` - Static files, Favicons, and SVG SVGs
- `/css/` - Component-based CSS styling (tokens, layout, sections, animations)
- `/js/` - Modular JavaScript files (Cursor interactions, loaders, scrolling observers, ambient sound)
- `index.html` - The core structure
